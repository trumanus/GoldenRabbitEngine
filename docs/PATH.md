# Development path

The canonical clone URL is the public GitHub repository (see `README.md`).

If you keep sources under a fixed disk layout:

- **Suggested local path:** `/sviluppo/GoldenRabbitEngine` (create `/sviluppo` on your machine if needed).
- **Bootstrap example (macOS/Linux):**

```bash
sudo mkdir -p /sviluppo && sudo chown "$USER" /sviluppo
git clone https://github.com/trumanus/GoldenRabbitEngine.git /sviluppo/GoldenRabbitEngine
```

If you initially cloned into `~/sviluppo/GoldenRabbitEngine`, move it:

```bash
mv ~/sviluppo/GoldenRabbitEngine /sviluppo/GoldenRabbitEngine
```
