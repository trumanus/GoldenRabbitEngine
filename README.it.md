# golden-rabbit-engine (Go) — README in italiano

Implementazione open source di **riferimento** per il **Volume 2** della saga Golden Rabbit: Registry, economia **K**, omeostasi, λ, memoria multi‑layer, conflitto, ACT, merge causale, sicurezza, outbox, maturità **M**, ecc.

> **Nota.** È una **libreria laboratorio**: API e test per i concetti, non uno stack produttivo completo (broker, firme, DB, OTel, executor LLM).

## Requisiti

- Go 1.22+

```bash
go test ./...
go run ./cmd/gr-demo
go run ./cmd/gr-full
```
